![image](https://github.com/user-attachments/assets/95e921c0-dbec-4010-a369-1fb5729467e7)**ID:** LMAO-001  
**Название:** Некорректное отображение текста на латинице в фигурных скобках.

**Описание:** Текст введенный в двойных фигурных скобках на латинице с любым регистром отображается некорректно в поле "Message Preview". 

**Шаги для воспроизведения:**
1. Перейти на страницу http://localhost:3000/
2. Кликнуть на кнопку "Message Editor"
3. Ввести в поле ввода "main" - "{{AbCd}}"
4. Кликнуть на кнопку "Preview"

**Ожидаемый результат:**
- В поле "Message Preview" должно отобразиться значение "{{AbCd}}".
